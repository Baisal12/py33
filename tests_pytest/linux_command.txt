cd-name - перемещает нас в какуюта папку(дерикторию)
ls - просматривает содержание папка
touch name - coздает фаил
mkdir name - создает папку

 
ls -a показы скрыты папки и фвилые
ls -l показывает фаилы и и папка с доп.ифой 
ls -la показ. файлы и папки со скрытыми файломи ипапаками + доп.информацию 


cd - перемещает в домашнию папку 
cd~ - тоже что cd 
cd . - перехотит в ту папку где находитесь 
cd .. - перехотит в передущую папку 
 
rm name - удаление файла name 
rmdir - удаляет пустую директорию
rm -rf - удаляет все 

	
history - вывод историю номанд 
histori число - выводит определеное количество 
histori | grep cлово - выводит все команды, в которой есть указ. слово 


pwd - пазывает абсалютный путь до текущей дириктории 
cat - выводит содержимое файла 
nano - текстовый редактор при помощи которего  мы можем записать данные в файл
